# Dashboard Page
A simple Dashboard home page using `HTML5`, `CSS3` and `Bootstrap 3`

## Installation
> Clone the repository using `git clone `
> Navigate inside the directory `dashboard-page`
> Open `index.html` to see the dashboard page
