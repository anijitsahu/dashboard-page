# Dashboard Page
A simple Dashboard home page using `HTML5`, `CSS3` and `Bootstrap 3`

## Installation

1. Clone the repository using `git clone `
2. Navigate inside the directory `dashboard-page`
3. Open `index.html` to see the dashboard page
